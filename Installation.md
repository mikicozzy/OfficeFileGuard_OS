**INSTALL**

Bisogna realizzare le seguenti azioni:

1. Copiare i file — CrocoOfficeFileReserveToggle.Cli.exe, CrocoOfficeFileGuard.Core.dll, CrocoIconOverlayShellExtension.dll, SharpShell.dll, lucchetto_tr.ico, DocumentFormat.OpenXml.dll, OutlookFileGuard_addin.dll e relativi file VSTO
2. Registrare il menu contestuale — scrivere le chiavi registry per .docx, .xlsx, .pptx che puntano al CLI
3. Registrare la shell extension COM — le chiavi registry per l'overlay icon che abbiamo già scritto nel .reg
4. Registrare l'Outlook add-in — scrivere le chiavi registry VSTO per l'add-in
