# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by *Keep a Changelog* and the project follows Semantic Versioning whenever practical.

## [1.0.0] - 2026-07-01

### Added

* Initial open-source release of OfficeFileGuard.
* Core library for managing the `reserved` Office document property.
* Command-line utility for toggling the reserved state of Office documents.
* Windows Explorer icon overlay integration.
* Microsoft Outlook VSTO add-in for detecting reserved Office documents attached to outgoing emails.
* Unit tests for the core library.
* Project documentation, architecture description, security policy, and contribution guidelines.

### Security

* Privacy-first design.
* Local-only document processing.
* No telemetry.
* No cloud services.
* No data collection.
